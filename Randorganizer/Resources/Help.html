What?

This app is an item, dungeon, and location tracker for the Zelda: A Link to the Past randomizer.

General Usage

This tracker is based heavily on the Stumpy tracker by Wolfman2000. It shares much of the logic and functionality. As of right now Randorganizer does not support keysanity or entrance/boss shuffles.

Items

Tap on an item to activate it. Continue to tap to either turn the item off or cycle through to the next, depending on the item in question.

Bosses / Dungeons
  • Tap on the chest to decrease the number of available items. The counts will reset after going to 0.
  • Tap on the black question mark circle to mark the reward you get for beating the dungeon. It can be (in order) a green pendant, a normal pendant, a normal crystal, or a pyramid (red) crystal.
  • Tap on the gold question mark circle to mark which medallion is required to enter the dungeon. It can be (in order) Bombos, Ether, or Quake.
  • Tap on the boss to mark the boss as defeated.
Map

The map has many squares on it: these represent locations where items are found. If it is an item outside of a dungeon, clicking on the square marks it as found. Dungeons cannot be marked as such: instead, refer to the boss guide above. Note that boss squares are double squares: the outer square represents the possibility of taking every treasure from the dungeon, while the inner square focuses solely on the boss of the dungeon.

Map Key

The color guide for the map is below.

Description Color
Available: This can be grabbed now.
Visible: This can be seen, but not taken.
Possible: This may be possible depending on other requirements.
Unavailable: This cannot be gotten at all.
Claimed: This has already been marked off.

If glitches or sequence breaks can be used to get items prematurely, the surrounding square color will be white instead of black.

Reset

This allows you to reset the game, clearing all items collected, locations visited, and dungeon information.

Default Bombs Selected

Enabling this option will default the bombs to selected when you reset or start a new game.
